# Portfolio Website

This repository contains my personal portfolio built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and a collection of custom React components.

## Features

- Responsive layout and navigation
- Case studies and project pages
- Contact section
- Styling with Tailwind CSS

## Tech Stack

- Next.js 14+
- React 19+
- TypeScript
- Tailwind CSS

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run the dev server

```bash
npm run dev
```

Then open:

- http://localhost:3000

## Build for Production

```bash
npm run build
```

### Start production server

```bash
npm run start
```

## Lint

```bash
npm run lint
```

## Project Structure (high level)

- `app/` - Next.js routes (App Router)
- `components/` - Reusable UI components
- `public/` - Static assets (PDFs, images, etc.)
